Folders and Files

For starters

You will need to set up the project folders and files like so
node_modules folder created by 'npm install'
routes folder
----- books.js // route definitions within routes
package.json // dependencies and project info
server.js // entry point of app, define server

Step 1 - Create package.json file

Make sure the following lines of code are present in the package.json file. Pay attention to the Node dependencies in particular. 
{
  "name": "hapi-rest-mongo",
  "version": "1.0.0",
  "description": "Simple REST project with hapijs and mongodb.",
  "main": "server.js",
  "private": true,
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  
  "dependencies": {
    "boom": "^4.2.0",
    "hapi": "^16.1.0",
    "joi": "^10.2.2",
    "mongojs": "^2.4.0",
    "node-uuid": "^1.4.7"
  }
}
